============
Installation
============

Default installation::

	$ curl -sL https://goo.gl/yy2UxF | bash

To install specific version::

	$ curl -sL https://goo.gl/yy2UxF | bash -ls install <version>

or::

	$ ~/.jrevolt/update.sh install <version>


------
Docker
------

::

	$ docker run jrevolt/launcher
